Photo naming convention

Responsive image sets
- `*-desktop.webp` is the desktop/default image.
- `*-tablet.webp` is the tablet image.
- `*-mobile.webp` is the mobile image.
- If a section only has one image, the desktop image can be reused across all breakpoints.

Current section image sets
- `about/about-history-{desktop,tablet,mobile}.webp`
- `academic/academic-curriculum-{desktop,tablet,mobile}.webp`
- `home/home-worship-spirit-{desktop,tablet,mobile}.webp`
- `partnership/partnership-sponsorship-{desktop,tablet,mobile}.webp`
- `school-life/school-life-sport-{desktop,tablet,mobile}.webp`

Hero images
- Hero assets follow the same breakpoint pattern.
- Use `*-hero-mobile.webp`, `*-hero-tablet.webp`, and `*-hero-desktop.webp`.
